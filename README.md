dokuwiki-plugin-openbadge
=========================

Provides a very simple way to host Open Badge Assertions and a UI for a user to collect them and put in there backpack in dokuwiki.

Usage example


```
---- openbadge ----
name       : Made First Android App Inventor App
eventtype  : workshop
image      : http://knowhow.access-space.org/lib/exe/fetch.php?media=badges:as-app-inventor-1st-app.png
description: Has made a simple Android App Using MIT Android App Inventor
criteria   : http://knowhow.access-space.org/doku.php?id=badges%3Amade_first_android_app_inventor_app
issuer_name: Access Space
contact    : martyn@access-space.org
recipients : martyn.eggleton@gmail.com
issued_on  : 2012-05-08
----
```
